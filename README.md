# ProductMetrics

Product metrics platform for tracking KPIs, analyzing trends, and making data-driven decisions.

**Live at [productmetrics.org](https://productmetrics.org)**

## About

ProductMetrics is a comprehensive reference for product metrics organized by the AARRR framework. Every metric includes definitions, formulas, benchmarks, and real-world examples.

## Stack

- **Framework**: Next.js (App Router)
- **Hosting**: Vercel
- **Styling**: Tailwind CSS
